PROJET SCOLAIRE — VERSION DESKTOP

GestTravaux Pro — JavaFX

GestTravaux Pro en version Java est une application desktop destinée à la gestion centralisée des travaux immobiliers. Elle est pensée pour les usages de pilotage, d’administration et de traitement complet des dossiers chantier avec une logique de bureau plus poussée.

Java JavaFX Hibernate MVC MySQL PDF

VUE D’ENSEMBLE

Rôle de la version Java

La version desktop est conçue pour une utilisation plus complète côté gestionnaire. Elle permet de manipuler les dossiers, structurer les informations, comparer les devis, gérer les entreprises et suivre l’ensemble des opérations liées à un chantier.

PRÉSENTATION

Architecture et logique de l’application

Objectif de la version Java

Cette application répond à un besoin de gestion métier plus approfondi que la version web. L’idée est d’offrir à l’utilisateur principal une interface de bureau permettant de consulter les dossiers, enregistrer les informations clés, traiter les devis, piloter les entreprises intervenantes et assurer un suivi structuré des étapes du chantier.

Socle technique

Langage : Java
Interface : JavaFX
Architecture : MVC
Persistance : Hibernate / JPA
Base de données : MySQL
Sorties : export ou génération de documents PDF

SYNTHÈSE

Ce que montre cette version

Apport fonctionnel

La version Java met davantage l’accent sur le pilotage des dossiers, la structuration des données et le traitement administratif des informations. Elle représente la partie la plus “gestionnaire” du projet GestTravaux Pro.

Apport technique

Ce projet valorise le développement orienté objet en Java, la construction d’interfaces desktop avec JavaFX, l’architecture MVC, la connexion à une base MySQL, la persistance avec Hibernate et la production de documents métiers.
