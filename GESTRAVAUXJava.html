PROJET SCOLAIRE — VERSION WEB

GestTravaux Pro — Symfony

GestTravaux Pro est une application web développée pour digitaliser la gestion de travaux immobiliers. Cette version Symfony est pensée pour les usages web et mobiles : suivi des chantiers par les inspecteurs, consultation des demandes par les entrepreneurs, accès ponctuel des propriétaires et administration distante.

Symfony 7 Doctrine ORM Twig MySQL Responsive PWA

VUE D’ENSEMBLE

Rôle de la version web

L’application web couvre les usages distants et terrain. Elle complète la version Java desktop, tout en partageant la même base de données MySQL afin de centraliser les informations liées aux biens, chantiers, devis, documents et acteurs du système.

PRÉSENTATION

Architecture et choix techniques

Objectif de l’application

Le projet répond à une problématique de gestion manuelle des travaux immobiliers : perte de temps, manque de traçabilité, échanges dispersés et suivi difficile de plusieurs chantiers en parallèle. La version web vise donc à fluidifier la circulation des informations, à faciliter la mobilité terrain et à permettre aux différents acteurs d’interagir avec le dossier chantier depuis une interface claire et accessible.

Socle technique

Framework : Symfony 7
ORM : Doctrine
Templates : Twig responsive
Base : MySQL partagée avec la version Java
Fonctions attendues : multi-rôles, validation, upload de fichiers
Approche : accès web / mobile avec logique orientée terrain

SYNTHÈSE

Ce que montre cette version

Apport fonctionnel

La version web de GestTravaux Pro apporte de la mobilité, une meilleure centralisation des échanges et une saisie plus directe des informations de chantier. Elle facilite la relation entre le terrain, les entrepreneurs et le suivi administratif du dossier.

Apport technique

Ce projet met en valeur l’utilisation d’un framework PHP moderne, la modélisation de données via ORM, la gestion de profils utilisateurs, les formulaires sécurisés, l’intégration de fichiers et l’organisation d’une application web connectée à une base commune avec une seconde application Java.
