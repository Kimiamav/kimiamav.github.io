Le mode débogage est une action qui sert déboguer ( donc vérifier) le code en cours d'exécution. Cette démarche permet d'arrêter l'exécution à des endroits désignés et de voir comment les choses progressent. 

Le point d'arrêt est un marqueur spéciale qui indique ou vous souhaiter arrêter l'exécution de l'application et quelques raccourcis clavier. 
Il existe trois types de point d'arrêt : 
	- les point d'arrêt de ligne,
	- les points d'observation de champ, 
	- les points d'arrêt de méthode
