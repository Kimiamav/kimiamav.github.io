GestTravaux Pro – Gestion de travaux immobiliers

Rôle : Développeuse – Projet SLAM 2e année (2025–2026)

Application complète pour la société fictive IMMOSYNC, permettant de gérer les chantiers de travaux immobiliers (création de chantier, appels d’offres, devis, suivi des travaux, réception, rapports).

  • Architecture hybride : application desktop Java pour les gestionnaires + application web Symfony (inspecteurs, entrepreneurs, propriétaires).
  • Base de données MySQL unique partagée (Hibernate côté Java, Doctrine côté Symfony).
  • Gestion du cycle de vie complet des chantiers, comparaison de devis, suivi d’avancement et génération de rapports PDF.
  • PWA pour les inspecteurs sur le terrain (photos, diagnostics, documents).
Symfony 7 Java / JavaFX MySQL Hibernate Doctrine ORM PWA Projet SLAM
